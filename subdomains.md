# Subdomínios

- Core: O que dá dinheiro, aquilo que não pode parar de jeito nenhum
- Supporting: Dá suporte para o core funcionar.
- Generic: Você precisa, mas não são tão importantes.

## Exemplos - E-commerce

### Core

- Compra
- Catálogo
- Pagamento
- Entrega

### Supporting

- Estoque

### Generic

- Notificação ao cliente
- Promoções
- Chat
